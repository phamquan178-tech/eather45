Eather45 Logo Assets

Copy these files into your Next.js project:

/public/images/eather45/logo-dark.svg
/public/images/eather45/logo-light.svg
/public/images/eather45/icon-dark.svg
/public/images/eather45/icon-light.svg

PNG fallbacks are included:
logo-dark.png
logo-light.png

Usage:
- logo-light.svg: use on dark backgrounds such as header, hero, footer.
- logo-dark.svg: use on light/off-white backgrounds such as DNA/brand sections.
- icon-light.svg / icon-dark.svg: use only when a compact mountain icon is needed.

Important:
Do not recreate the logo with CSS text or hand-drawn SVG in components.
Use the asset file directly so the logo stays consistent.
